## [0.2.0] - 2026-02-19
### Added
- Cmd+Q keyboard shortcut to quit the app.
- Window visible on all workspaces (Spaces).

### Changed
- Migrated from modular CSS to Tailwind CSS v4.
- Updated app identifier.

## [0.1.0] - 2025-11-19
### Added
- Initial public release.
- macOS global shortcut for instant launch (Cmd+Shift+M).
- New memo creation shortcut (Cmd+N).
- Auto-focus on text area when the window appears.
- Hide window with Esc while keeping the current text.
- System tray integration with Show/Quit.
